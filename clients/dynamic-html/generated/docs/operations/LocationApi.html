This is the API

locationAreaList

Location areas are sections of areas, such as floors in a building or cave. Each area has its own set of possible Pokémon encounters.

URL

https://pokeapi.co/api/v2/location-area/

HTTP Method

GET

Response Type

PaginatedLocationAreaSummaryList

Parameters

  • Query: limit Integer

    Number of results to return per page.

  • Query: offset Integer

    The initial index from which to return the results.

locationList

Locations that can be visited within the games. Locations make up sizable portions of regions, like cities or routes.

URL

https://pokeapi.co/api/v2/location/

HTTP Method

GET

Response Type

PaginatedLocationSummaryList

Parameters

  • Query: limit Integer

    Number of results to return per page.

  • Query: offset Integer

    The initial index from which to return the results.

  • Query: q String

    > Only available locally and not at [pokeapi.co](https://pokeapi.co/docs/v2) Case-insensitive query applied on the `name` property.
